# AIM ACADEMY — adaptive-warfare aim trainer

An FPS aim-training game set inside photorealistic battlegrounds generated by
the **World Labs Marble API** (Gaussian splats + collider meshes), with
humanoid robot opponents that **adapt between matches** — their aim, reflexes
and tactics evolve from how you play, and they keep what they learned forever.

## Run

```bash
# .env must contain: WLT_API_KEY=<your key>  (server fallback key)
node server.mjs
```

Open **http://localhost:5173** — Node 18+, zero npm dependencies.

## Features

- **AI-generated maps**: pick a battlefield, or type any idea and it gets
  "battle-ized" into a war zone via the Marble API (hero or draft quality).
- **Bring-your-own-key**: players can connect their own World Labs API key
  (session-only) to load the worlds from their account and forge maps on
  their own credits. Maps use the API's `metric_scale_factor` for true scale.
- **Adaptive opponents**: population-based evolution — each bot mutates a
  shared genome (aim / reflexes / mobility / evasion / aggression); after
  every round the genome shifts toward what hurt you most. Persisted in
  `localStorage` with full match history, across sessions.
- **Real combat behavior**: state-machine bots (move → acquire → fight),
  procedural skeletal animation on the biped robot rig (walk cycle, two-handed
  aim stance, reload pump, recoil), line-of-sight and cover via the world's
  collider mesh + BVH raycasts, random cover props, headshots, COD-style regen.
- **Cinematic death**: K.I.A. video sequence with black/red fade.
